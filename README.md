## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 用手機開啟此網站

- **本機測試（同一 Wi‑Fi）**：`localhost` 在手機上指的是手機自己，所以用手機無法打開電腦的 `http://localhost:3000`。請在電腦終端執行 `ipconfig`（Windows）或 `ifconfig`（Mac/Linux），查到你電腦的區域 IP（例如 `192.168.1.100`），然後在手機瀏覽器輸入 **`http://192.168.1.100:3000`**（把 IP 換成你的），且手機與電腦要在同一個 Wi‑Fi 下。
- **正式上線**：部署到 [Vercel](https://vercel.com) 或其它主機後，會得到一個網址（例如 `https://xxx.vercel.app`），用手機開這個網址即可；建議使用 HTTPS，較少被瀏覽器阻擋。

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
